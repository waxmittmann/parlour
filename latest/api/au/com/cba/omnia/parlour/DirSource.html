DirSource

Related Doc: package parlour

class DirSource extends FixedPathSource with TextLineScheme

A directory representation that help with Source/Sink match based on merely the path. The TextLineScheme although mixed in, would be ignored as sqoop handles paths directly i.e. not through any passed in Tap.
